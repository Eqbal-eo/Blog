1. Home Page (موقع المدونة الرئيسي)
   - عرض التدوينات الأخيرة
   - رابط لتسجيل الدخول / التسجيل
   - عرض أحدث المقالات
   - روابط التنقل الرئيسية

2. Login Page (صفحة تسجيل الدخول)
   - نموذج تسجيل الدخول (اسم المستخدم، كلمة المرور)
   - رابط للتسجيل إذا لم يكن لديك حساب

3. Register Page (صفحة التسجيل)
   - نموذج لإنشاء حساب جديد (اسم المستخدم، كلمة المرور)
   - رابط لتسجيل الدخول إذا كان لديك حساب بالفعل

4. Dashboard (لوحة تحكم المدير)
   - مرحبًا بالمدير (عرض اسم المستخدم)
   - روابط:
     - إنشاء تدوينة جديدة
     - تعديل تدوينة
     - حذف تدوينة
     - تسجيل الخروج
   - عرض التدوينات الخاصة بالمدير
   - إحصائيات وملخص النشاط

5. Create Post Page (صفحة إنشاء التدوينة)
   - نموذج لإنشاء تدوينة جديدة (عنوان، محتوى)
   - محرر نصوص متقدم
   - إمكانية رفع الصور
   - زر للنشر

6. Edit Post Page (صفحة تعديل التدوينة)
   - نموذج لتحرير التدوينة (عنوان، محتوى)
   - زر لحفظ التعديلات

7. Select Post to Edit (اختيار التدوينة للتعديل)
   - قائمة بالتدوينات الخاصة بالمدير
   - رابط لكل تدوينة لتحريرها

8. Select Post to Delete (اختيار التدوينة للحذف)
   - قائمة بالتدوينات الخاصة بالمدير
   - رابط لحذف كل تدوينة مع تأكيد الحذف

9. About Page (صفحة حول الموقع)
   - معلومات عن المدونة
   - رؤية ورسالة الموقع
   - معلومات الاتصال

10. Blogs Page (صفحة المدونات)
    - عرض جميع المقالات
    - إمكانية التصفية والبحث
    - ترتيب حسب التاريخ أو الشعبية

11. Author Page (صفحة الكاتب)
    - الملف الشخصي للكاتب
    - قائمة مقالات الكاتب
    - معلومات التواصل

12. Contact Page (صفحة الاتصال)
    - نموذج الاتصال
    - معلومات التواصل
    - خريطة الموقع (إن وجدت)

13. Settings Page (صفحة الإعدادات)
    - تعديل معلومات الحساب
    - تغيير كلمة المرور
    - تفضيلات المستخدم

14. Post Page (صفحة المقال)
    - عرض المقال كاملاً
    - معلومات الكاتب
    - التعليقات
    - أزرار المشاركة
